PoseMagic is designed to be compatible with AVsitter workflows.

## Notecards
Your \AVpos\ notecard is 100% compatible. Just rename it to \PMposes\.

## Script Mapping
| AVsitter | PoseMagic |
| :--- | :--- |
| \[AV]sitA\ | \! PM - core\ (per sitter) |
| \[AV]sitB\ | \! PM - dbc\ (per sitter) |
| \[AV]helper\ | \! PM - anchor\ |
