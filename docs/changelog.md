## Version 1.0

**Initial Release**

- Complete rewrite of AVsitter2
- Unlimited poses support (via expandable memory)
- Multiple notecard support (PMposes, PMposes 1, etc.)
- Unlimited props
- Unlimited sitters (with smart pagination)
- Linkset Data integration for better memory management
- 100% AVpos notecard compatibility
- Personal adjustments system
- All core AVsitter features maintained

### Key Improvements Over AVsitter

- **No 230-250 pose limit** - Expandable memory system
- **No single notecard limit** - Multiple notecards supported
- **No 100 prop limit** - Unlimited props
- **No ~10 sitter limit** - Unlimited sitters with pagination
- **Better memory management** - Reduced stack-heap errors
- **Linkset Data support** - Modern memory architecture
